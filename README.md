# Shanil Wijesinghe — Portfolio

Minimal portfolio site built with **Next.js (App Router)**, **Tailwind CSS**, and
**shadcn/ui**-style components. Designed to deploy on **Vercel**.

## Getting started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Structure

- `app/page.tsx` — home page with the 3×2 grid of selected work.
- `app/work/[slug]/page.tsx` — per-project case-study page (gallery + Strategy /
  Design Method / Elements of the Design sections).
- `lib/projects.ts` — single source of truth for all project content. Edit titles,
  blurbs, image paths, and section copy here.
- `components/` — UI building blocks (`ui/` holds the shadcn-style primitives).

## Images

Placeholder artwork lives in `public/placeholders/` and is generated by
`scripts/generate-placeholders.mjs` (`node scripts/generate-placeholders.mjs`).
Drop real images into `public/` and update the `thumbnail` / `gallery` paths in
`lib/projects.ts` when they're ready.

## Deploy

Push to GitHub and import the repo in Vercel — no extra configuration needed.
